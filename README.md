## Face_Know
###
####  一 人脸识别以及配准
功能简述：使用Opencv numpy 实现简单的人脸识别采集，然后通过根据先前采集的人脸信息进行实时的人脸识别以及人脸配准。实时视频使用手机摄像头通过局域网进行传输。

    第三库：Opencv numpy 
    传输软件： IP摄像头

##### 运行文件为:video.py

##### 运行简述：
1. 当人脸采集很多过后，每次识别器训练的时间会过长。然后，人脸匹配速度下降。
2. 人脸匹配过程中，进行匹配时，帧率下降。
3. 量性分析时，识别准确率在80%左右。较为准确。
4. 当识别完成后，按Q停止运行。
